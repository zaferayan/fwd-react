# Egzersiz #1 — Temel Tipler & Fonksiyonlar

## Görev 1
Aşağıdaki fonksiyona doğru tip tanımlarını ekle. Fonksiyon iki sayı alıp çarpımını döndürsün.

```typescript
function carp(a, b) {
  return a * b;
}
```

## Görev 2
`carp("5", 3)` çağrısını yap ve TypeScript'in hata verdiğini gözlemle. Sonra düzelt.

## Görev 3
Bir kullanıcının adını ve **isteğe bağlı** yaşını alan, `string` döndüren bir fonksiyon yaz.

- Yaş verilmişse → `"Ali, 25 yaşında"`
- Yaş verilmemişse → `"Ali"`
