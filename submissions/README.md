# 📬 Submissions

Bu klasör, bootcamp bitirme projesi teslimlerini içerir. Programı tamamlamak ve sertifika almaya hak kazanmak için bitirme projenizi buraya teslim edin.

---

## Nasıl Teslim Edilir?

### 1. Repo'yu Fork'layın

GitHub üzerinden [bu repo'yu](https://github.com/Acadevia-mg/fwd-react) fork'layın.

### 2. Fork'unuzu Klonlayın

```bash
git clone https://github.com/<KULLANICI-ADINIZ>/fwd-react.git
cd fwd-react
```

### 3. Klasörünüzü Oluşturun

`submissions/` altında kendi adınızla bir klasör oluşturun:

```bash
mkdir -p submissions/ad-soyad
```

> Örnek: `submissions/furkan-unsalan`

### 4. Projenizi Ekleyin

Bitirme projesi dosyalarınızı kendi klasörünüze ekleyin:

```
submissions/furkan-unsalan/
├── src/
├── public/
├── package.json
├── README.md         ← Projenizi açıklayan kısa bir README
└── ...
```

### 5. Commit ve Push

```bash
git add .
git commit -m "feat: furkan-unsalan bitirme projesi"
git push origin main
```

### 6. Pull Request Açın

1. GitHub'da fork'unuza gidin
2. **"Contribute" → "Open pull request"** butonuna tıklayın
3. PR başlığı: `[Bitirme Projesi] Furkan Ünsalan`
4. Açıklama kısmında projeniz hakkında kısa bir bilgi yazın
5. PR'ı gönderin ve review sürecini bekleyin

---

## Kurallar

- Sadece **kendi klasörünüzde** değişiklik yapın
- Başkalarının klasörlerini düzenlemeyin
- Klasör adınızı `ad-soyad` formatında, küçük harfle ve Türkçe karakter kullanmadan yazın
- Projenize ait bir `README.md` eklemeyi unutmayın
